# FAIID_RasPI5
A version of the FAIID project for the Raspberry Pi 5

NOTE for team members:
I want to keep this project as organized as possible. So, there are three directories for this project; src, include, and lib.
Any code written by YOU and you alone shouuld be in a .c file in the src directory. These .c files should also have a corresponding .h file in the include directory.
If you are including any code written by someone else it should go into the lib directory; this includes any automatically generated utilities, drivers, etc.
Finally, when adding files of any kind make sure they are listed in your commit message so I can add them to the make file.

Thanks, and happy coding.
